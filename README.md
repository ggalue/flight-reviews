# README

## Some of the features of this app include:

- Ruby version: 2.6.5
- Rails version: 6.0.3
- Database: postgresql
- React version: 16.12.0
- React Hooks API

## Running it locally
- run bundle exec rails db:prepare
- run yarn install
- run bundle install
- run rails db:create
- run rails db:seed
- run bundle exec rails s
- navigate to http://localhost:3000
